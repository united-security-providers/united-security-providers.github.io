Overview

You will find the following documentation here:

  • USP Core WAAP (Web Application and API Protection) provides secure access to web-based applications and resources, while simplifying the process of configuration and deployment.
  • USP Core Authenticate provides secure authentication to web-based applications and resources. It runs an instance of the "USP Secure Login Service (SLS)".
  • The USP Network Authentication System© (USP NAS) is a network access control solution (NAC) which provides visibility of and secure access to your network.
